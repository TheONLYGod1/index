Moderation Commands:

Moderation - 

Kick : Need to kick someone? 
*kick
Ban: Need to ban someone?
*ban
Report: Need to report someone?
(Must create a channel named "warnings")
*report
Need to lock a channel?
*lockdown

Music Commands:

Play : Want to listen to music?
*play
NP : Shows whats currently playing
*np
Volume : Need to change the volume?
*volume
Skip : Skips the song
*search
Queue : See what songs are playing and are next
*queue
Pause : Pause the song thats playing
*pause
Resume : Resume the song that was paused
*resume
Stop : Stop the song
*stop
